# 📊 PULSO_2026_ELECCIONES

Repositorio que contiene bases de datos y documentación relacionadas con los resultados oficiales de las Elecciones Regionales y Municipales 2022 en el Perú.

La información ha sido recopilada y sistematizada a partir de los repositorios oficiales de la Oficina Nacional de Procesos Electorales (ONPE).

## 📌 Fuente de datos

Los datos provienen del portal de Datos Abiertos de la ONPE:

### Documento oficial:
https://www.onpe.gob.pe/modElecciones/Datos-abiertos-Historico-resultados/ERM-2022.pdf
